Ratio 值的雜訊防線

每個 ratio 從原始光譜到告警狀態之間，會通過 八道濾網。每一道擋掉不同性質的雜訊 —— 隨機抖動靠平滑與門檻放寬壓制，near-noise 的無意義比值整條剔除，暫態突波靠時間確認過濾。雜訊不是被單一步驟消滅，而是被逐層量化、放寬、剔除或降權。

1

Extraction

擷取層 · 把線強度量乾淨

LineIntensityExtractor.Extract → LineMeasurement LineRegion.PeakSearchHalfWidthNm

每條線 = 峰/積分減去兩側 baseline 窗；同時從 baseline 窗估一個局部連續雜訊 σ，換算成 SNR，讓後面每一層都知道這個數字有多可信。每幀在 ±HalfWidth 內重新對準真正的峰，波長校正漂移不會把讀值推到窗邊、混進背景。

處理 波長/背景漂移
2

Smoothing

平滑層 · 壓隨機抖動

EMA α = 1 − exp(−dt/τ), τ = EmaTauSeconds (5s) _emaVar (EWMA 變異數) → LiveSigma

比值先過時間常數式 EMA：用 dt 而非固定權重，幀率不均、掉幀時平滑強度仍正確。告警吃的是平滑後的 _ema，不是單幀原始值。同時用 EWMA 變異數量測「現在這條 ratio 有多吵」，餵給第 4 層與 leak-rate。

處理 隨機抖動
3

SNR gate

訊號品質閘 · near-noise 直接不採信

MinSnr (5) → RatioState.LowSignal Golden Run 同閘 · MinBaselineAcceptFraction (0.5)

signal 或 reference 任一線 SNR < 門檻 → 整條 ratio 移出合成告警、也踢出 leak-rate 擬合。兩條貼近雜訊的線相除毫無意義，再怎麼平滑都是垃圾。基線擷取用同一個閘，且通過幀數不足一半時整條拒收，防止弱線偏出假基線。

處理 near-noise 無意義
4

Adaptive thresholds

門檻放寬 · 隨雜訊自動抬高

EffectiveSigma = max(GoldenRun σ, LiveSigma)

雙層門檻取「基線當時的雜訊」與「現在的即時雜訊」較大者放寬。訊號比基線更吵時門檻自動抬高 —— ratio 不會被自己的抖動絆倒。

Warn = max(WarnFactor·μ, μ + 3σ) Alarm = max(AlarmFactor·μ, μ + 6σ)
處理 隨機抖動
5

Sustained confirm

時間確認 · 濾掉暫態尖峰

ConfirmSeconds (15s) 持續超標才升級 latched alarm

_ema 必須持續超過門檻達 15 秒才升級狀態；單一幀或幾秒的突波不會升警。確認後閂住 (latch)，直到操作者 Acknowledge —— 訊號在門檻邊緣抖動不會造成告警閃爍。

處理 暫態突波
6

Grace window

邊界防閃 · 別在 floor 上抖動

IdleGraceSeconds (1.5s) plasma floor · SNR floor

單幀掉到 plasma floor 或 SNR floor 以下時，先保持目前的 active 狀態 1.5 秒，避免橫幅以幀率在「OK ↔ 等待電漿 / 低訊號」之間閃爍。熄電漿時保留 latch，但重啟平滑與趨勢，讓舊值不會跨過空窗橋接進來。

處理 邊界閃爍
7

Absolute mode · 新增

弱線 σ 正規化 · 取代小分母

顯示 100 + (20/SigmaWarn)·(ema−baseMean)/EffectiveSigma leak-rate 擬合 Δ = value − baseMean

絕對強度模式下 signal 線基線 mean 貼近雜訊底，任何 value ÷ baseline 都會大幅擺盪。改用雜訊 σ 正規化顯示、用絕對增量 Δ 算 leak-rate（σ 直接取 √emaVar），熱路徑上不再除小分母 —— baseline 仍落在 100，一個 warn-σ 落在 120。

處理 near-noise 無意義 隨機抖動
8

Leak-rate fusion

融合層 · 用雜訊當權重

LeakRateEstimator.Estimate 反變異數加權 w = 1/σ_x² 跨比值一致性 → confidence

每條 ratio 的 σ_x（來自 √emaVar）決定它在反變異數加權融合中的份量 —— 越吵的 ratio 權重越低。多條 ratio 一致性再換算成 confidence，χ² 越大信心越低。雜訊在這裡不是被濾掉，而是被量化並用來降權。
